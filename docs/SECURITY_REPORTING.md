# Security Reporting

TruthShield should provide a responsible disclosure path before public launch.

## Report Scope

- Authentication or token handling bugs.
- Admin authorization bypasses.
- Evidence URL SSRF or unsafe redirect issues.
- Payment callback signature validation issues.
- Extension origin or iframe message handling issues.

## Initial Response Target

- Acknowledge reports within 72 hours.
- Triage severity within 7 days.
- Publish a public summary after remediation when disclosure is safe.

## Private Data

Do not include real user tokens, payment identifiers, or private evidence URLs in public issue trackers.
